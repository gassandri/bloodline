BLOODLINE · App de entrenamiento (Familia)
========================================

Archivos:
- index.html
- styles.css
- data.js   (planes base Semana 1)
- app.js    (lógica: splash 3s, perfiles, semanas, timer, progreso, admin)
- logo-bloodline.png

Cómo abrir (Windows, sin programar):
1) Copiá la carpeta al Escritorio (si querés).
2) Abrí index.html con Chrome/Edge y listo.

Recomendado (para evitar bloqueos del navegador con localStorage):
- Abrí un servidor local:
  - Abrí CMD en la carpeta
  - Ejecutá:  python -m http.server 8000
  - Abrí: http://localhost:8000
